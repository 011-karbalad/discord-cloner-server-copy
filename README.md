## 📦 قابلیت‌ها

- 🎯 گرفتن بکاپ کامل از ساختار سرور:
  - رول‌ها (Roles)
  - کتگوری‌ها (Categories)
  - چنل‌ها (Channels)
- ♻️ ریستور کامل بکاپ در همان سرور یا سرور دیگر
- ذخیره بکاپ به صورت فایل JSON قابل انتقال

---

## 🛠️ تکنولوژی‌ها

- Python `3.10+`
- کتابخانه `discord.py` نسخه 2 یا بالاتر
- ماژول‌های استاندارد `json` و `os`
